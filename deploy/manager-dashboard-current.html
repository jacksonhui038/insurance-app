🔗 合併代理人

當代理人打錯名字產生重複時，可將兩個記錄合併為一個。所有數據（客戶、保單、佣金等）會合併到目標代理人。

🗑️ 刪除代理人

適用於代理人離職或入錯名。刪除後該代理人嘅所有數據（客戶、保單、佣金等）會從經理端移除。

🔑 重置代理人 PIN

適用於代理人忘記 PIN 或無法登入。重置後代理人可用新 PIN 登入。

🔧 補填職級 / 所屬經理

為舊帳號補填職級同所屬經理，自動計算層級鏈（hierarchyPath），同時更新 manager 欄位。

👤 註冊經理 / 代理人

將新嘅經理或代理人加入 Supabase 雲端，設定佢嘅職級同所屬上司。
